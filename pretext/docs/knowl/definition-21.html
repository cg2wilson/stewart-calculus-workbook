Definition 2.1.8. The Derivative (at \(x=a\)).

The derivative of a function \(f\) at input \(x=a\) is given by

\begin{equation*} f'(a) = \lim_{h\to 0} \dfrac{f(a+h)-f(a)}{h} \end{equation*}

or

\begin{equation*} f'(a) = \lim_{x\to a} \dfrac{f(x)-f(a)}{x-a} \end{equation*}

if this limit exists.
