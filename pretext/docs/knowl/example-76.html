  4. \(\displaystyle \ds \lim_{x\to -1}\left[f(x)g(x)\right]\)

  5. \(\displaystyle \ds \lim_{x\to -1}\left[-2f(x)+3g(x)\right]\)

  6. \(\displaystyle \ds \lim_{x\to 2} [3f(x)]\)
