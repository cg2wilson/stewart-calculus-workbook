Item 4.
If we set \(u=\sin x\) and \(dv = \cos x\, dx\text{,}\) we get
\begin{equation*} \ds \int \sin x\cos x\, dx = \dfrac{1}{2}\sin^2x + C \end{equation*}
However, if we set \(u=\cos x\) and \(dv = \sin x\, dx\text{,}\) we get
\begin{equation*} \ds \int \sin x\cos x\, dx = -\dfrac{1}{2}\cos^2x + C \end{equation*}
