Item a.
Begin by sketching the region. When we do this, we find that we can write
\begin{equation*} R = \lrbrace{(x,y)\mid 0\leq x\leq 1, 0\leq y\leq \sqrt{1-x^2}} \end{equation*}
This means we're looking at a Type I integral, which can be evaluated by computing
\begin{equation*} \int_0^1\int_0^{\sqrt{1-x^2}} x^2 + 2y\, dy\, dx \end{equation*}
which gives \(\dfrac{\pi}{8} + \dfrac{4}{3}\)
