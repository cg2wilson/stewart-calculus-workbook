Solution 1.3.1.1.
  1. \((f+g)(x) = \sqrt{3-x}+\sqrt{x^2-1}\) with domain \((-\infty,-1]\)
  2. \((f-g)(x) = \sqrt{3-x}-\sqrt{x^2-1}\) with domain \((-\infty,-1]\)
  3. \((fg)(x) = \sqrt{-x^3+3x^2+x-3}\) with domain \((-\infty,-1]\)
  4. \((f/g)(x) = \sqrt{\dfrac{3-x}{x^2-1}}\) with domain \((-\infty,-1)\)
