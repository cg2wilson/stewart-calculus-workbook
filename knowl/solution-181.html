Solution 2.3.34.1.

  1. \(\displaystyle \dfrac{dy}{dx} = 3x^2f(x) + x^3f'(x)\)
  2. \(\displaystyle \dfrac{dy}{dx} = \dfrac{xf'(x)-f(x)}{x^2}\)
  3. \(\displaystyle \dfrac{dy}{dx} = \dfrac{2xf(x)-x^2f'(x)}{[f(x)]^2}\)
  4. \(\displaystyle \dfrac{dy}{dx} = -\dfrac{1}{2}x^{-3/2} + \dfrac{1}{2}x^{-1/2}f(x) + x^{1/2}f'(x)\)
