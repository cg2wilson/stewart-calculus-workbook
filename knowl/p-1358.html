  1. \(\dfrac{dy}{dx} = \dfrac{\cos\theta(1+2\sin\theta)}{(1+\sin\theta)(1-2\sin\theta)}\text{,}\) so
    \begin{equation*} \dfrac{dy}{dx}\bigg\rvert_{\theta = \pi/6} = \dfrac{\sqrt{3}}{0} \end{equation*}
    which is undefined.
  2. The tangent line is horizontal when \(\cos\theta(1+2\sin\theta) = 0\iff \theta = \dfrac{\pi}{2},\dfrac{7\pi}{6},\dfrac{11\pi}{6}\text{.}\)
    The tangent line is vertical when \((1+\sin\theta)(1-2\sin\theta) = 0\iff \theta = \dfrac{3\pi}{2},\dfrac{\pi}{6},\dfrac{5\pi}{6}\)
