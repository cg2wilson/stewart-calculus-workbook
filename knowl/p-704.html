The expression \(\dfrac{\tan x - x}{x^3}\) is a \(0/0\) indeterminate form, so we may use l'Hospital's Rule. Applying it gives
\begin{equation*} \ds \lim_{x \to 0} \dfrac{\tan x - x}{x^3} \overset{LH}{=} \lim_{x\to 0} \dfrac{\sec^2x - 1}{3x^2} \end{equation*}
Attempting to evaluate gives another \(0/0\) indeterminate form, so applying l'Hospital's Rule again gives
\begin{equation*} \ds \lim_{x\to 0} \dfrac{\sec^2x -1}{3x^2} \overset{LH}{=} \lim_{x\to 0} \dfrac{2\sec^2x\tan x}{6x} \end{equation*}
Again we have a \(0/0\) indeterminate form, so apply the Rule a third time:
\begin{equation*} \ds \lim_{x\to 0} \dfrac{2\sec^2x\tan x}{6x} \overset{LH}{=} \lim_{x\to 0} \dfrac{4\sec^2x\tan x + 2\sec^4x}{6} \end{equation*}
Now we can resolve the limit as \(\dfrac{0 + 2}{6} = \dfrac{1}{3}\)
