	\subsubsection*{Summary of Graphing}
	\addcontentsline{toc}{subsubsection}{Summary of Graphing}
		We now have a guide for how to sketch the graph of most functions.
		\begin{rmk}[Guidelines for Curve Sketching]
			\showto{ins}{
				\begin{enumerate}[(1)]
					\item Find the domain of the function.
					\item Identify $x-$ and $y-$intercepts.
					\item Locate symmetry in the graph, i.e. determine if the function is even, odd, periodic, etc.
					\item Locate any asymptotes$-$vertical, horizontal, or slant.
					\item Determine the intervals where the function is increasing or decreasing.
					\item Find local maxima/minima.
					\item Determine the intervals where the function is concave up/concave down.
				\end{enumerate}	\\
			}
			\showto{st}{
