	\subsubsection*{The Rule for Indefinite \& Definite Integrals}
	\addcontentsline{toc}{subsubsection}{The Rule for Indefinite \& Definite Integrals}
		The \emph{substitution rule} for integrals allows us to take antiderivatives of composite functions.
		\begin{thm}[Substitution Rule]
			If $u = g(x)$ is a differentiable function whose range is an interval $I$ and $f$ is continuous on $I$, then
			\showto{ins}{
				\[\int f(g(x))g'(x)\,dx = \int f(u)\,du\]
			}
			\showto{st}{
				\vspace{.5in}
			}
		\end{thm}
		 	\vs{.25}

			For symmetric functions, we have the following properties:\\

				\begin{enumerate}[(1)]
				\setlength\itemsep{5pt}
					\item If $f$ is continuous on $[-a,a]$ and $f$ is even, then \fbox{$\ds \int_{-a}^a f(x)\,dx = 2\int_0^a f(x)\,dx$}
